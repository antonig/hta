Just some exercises to rediscover the forgotten art of hta...

crono:      
chronometer on screen using document.setinterval

factoriza2: 
prints factors of entered number up to 10^16-1  or declares it a prime. If number not prime it fins next prime up and down. This last function mqy give timeout errors as it's processor intensive.

15game:     
the old puzzle by Sam Lloyd driven by mouse clicks on a html table

simple_calc:
Very simple calculator exeercise where user keys in an expression and clicks on calculate. No keys on screen. It relies on vbscript's eval so not safe. All functions and operators in vbs are available
